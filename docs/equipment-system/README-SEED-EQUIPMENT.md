# วิธีนำเข้าข้อมูลอุปกรณ์ตัวอย่าง

มี 3 วิธีในการนำเข้าข้อมูลอุปกรณ์ตัวอย่างเข้าสู่ระบบ:

## วิธีที่ 1: ใช้ Browser Console (แนะนำ - ง่ายที่สุด)

1. เปิดเว็บแอปพลิเคชันในเบราว์เซอร์
2. ล็อกอินด้วยบัญชี Admin
3. ไปที่หน้า "จัดการอุปกรณ์" (`/admin/equipment`)
4. เปิด Developer Console (กด F12 หรือ Ctrl+Shift+I)
5. คัดลอกโค้ดจากไฟล์ `scripts/seed-equipment-data-simple.js`
6. วางโค้ดใน Console แล้วกด Enter
7. รอให้สคริปต์ทำงานเสร็จ (จะแสดงผลลัพธ์ใน Console)
8. รีเฟรชหน้าเพื่อดูข้อมูลที่เพิ่มเข้ามา

## วิธีที่ 2: ใช้ Node.js Script

### ข้อกำหนด:
- Node.js ติดตั้งแล้ว
- Firebase configuration ตั้งค่าใน `.env` แล้ว

### ขั้นตอน:

1. ติดตั้ง dependencies:
```bash
npm install firebase
```

2. รันสคริปต์:
```bash
node scripts/seed-equipment-data.js
```

3. รอให้สคริปต์ทำงานเสร็จ
4. เปิดเว็บแอปพลิเคชันและตรวจสอบข้อมูล

## วิธีที่ 3: เพิ่มทีละรายการผ่าน UI

1. เปิดเว็บแอปพลิเคชัน
2. ล็อกอินด้วยบัญชี Admin
3. ไปที่หน้า "จัดการอุปกรณ์"
4. คลิกปุ่ม "เพิ่มอุปกรณ์"
5. กรอกข้อมูลตามตัวอย่างใน `data/sample-equipment.json`
6. บันทึกข้อมูล

## โครงสร้างข้อมูล

แต่ละรายการอุปกรณ์ประกอบด้วย:

- `equipmentNumber`: หมายเลขครุภัณฑ์ (ต้องไม่ซ้ำ)
- `name`: ชื่ออุปกรณ์
- `category`: ประเภทอุปกรณ์ (object)
  - `id`: รหัสประเภท
  - `name`: ชื่อประเภท
  - `icon`: ไอคอน
- `brand`: ยี่ห้อ
- `model`: รุ่น
- `description`: รายละเอียด
- `specifications`: ข้อมูลจำเพาะทางเทคนิค (object)
- `status`: สถานะ (active, maintenance, retired, lost)
- `location`: สถานที่ตั้ง (object)
  - `building`: อาคาร
  - `floor`: ชั้น
  - `room`: ห้อง
  - `description`: รายละเอียดเพิ่มเติม
- `purchaseDate`: วันที่ซื้อ
- `purchasePrice`: ราคาซื้อ (บาท)
- `vendor`: ผู้จำหน่าย
- `warrantyExpiry`: วันหมดประกัน
- `tags`: แท็ก (array)
- `notes`: หมายเหตุ

## การตรวจสอบข้อมูล

หลังจากนำเข้าข้อมูลแล้ว ให้ตรวจสอบ:

1. ✅ ข้อมูลแสดงในหน้า "จัดการอุปกรณ์"
2. ✅ สามารถค้นหาอุปกรณ์ได้
3. ✅ สามารถกรองตามประเภทได้
4. ✅ สามารถแก้ไขข้อมูลได้
5. ✅ สามารถดูรายละเอียดได้

## การลบข้อมูลทดสอบ

หากต้องการลบข้อมูลทดสอบ:

1. ไปที่ Firebase Console
2. เลือก Firestore Database
3. เลือก collection `equipmentManagement`
4. ลบ documents ที่มี `equipmentNumber` ขึ้นต้นด้วย `EQ-2024-`

หรือใช้ UI:
1. เปิดหน้า "จัดการอุปกรณ์"
2. คลิกที่อุปกรณ์ที่ต้องการลบ
3. คลิกปุ่ม "ลบ"

## หมายเหตุ

- ข้อมูลตัวอย่างไม่มีรูปภาพ (สามารถเพิ่มภายหลังได้)
- หมายเลขครุภัณฑ์ต้องไม่ซ้ำกัน
- วันที่จะถูกแปลงเป็น Firestore Timestamp อัตโนมัติ
- ข้อมูล `createdBy` และ `updatedBy` จะถูกตั้งค่าเป็น UID ของผู้ใช้ที่รันสคริปต์

## การแก้ไขข้อมูล

หากต้องการแก้ไขข้อมูลตัวอย่าง:

1. แก้ไขไฟล์ `data/sample-equipment.json`
2. รันสคริปต์ใหม่อีกครั้ง (ต้องลบข้อมูลเก่าก่อน)

หรือแก้ไขผ่าน UI:
1. เปิดหน้า "จัดการอุปกรณ์"
2. คลิกปุ่ม "แก้ไข" ที่อุปกรณ์ที่ต้องการ
3. แก้ไขข้อมูล
4. บันทึก
