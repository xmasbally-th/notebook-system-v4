## 1. บทบาทและสิทธิ์ของ Staff

เจ้าหน้าที่ (Staff) ทำหน้าที่ดำเนินการด้านการยืม-คืนอุปกรณ์ในระดับปฏิบัติการ:

| สิทธิ์ | รายละเอียด |
|--------|-----------|
| อนุมัติ/ปฏิเสธคำขอยืม | ดำเนินการกับคำขอจากผู้ใช้ |
| รับคืนอุปกรณ์ | บันทึกการคืนอุปกรณ์ในระบบ |
| ดูอุปกรณ์ค้างคืน | ติดตามรายการที่เลยกำหนด |
| จัดการการจอง | อนุมัติ/ปฏิเสธการจองล่วงหน้า |
| ยืมอุปกรณ์ (อนุมัติอัตโนมัติ) | Staff ยืมได้โดยไม่ต้องรออนุมัติ |

> ⚠️ Staff **ไม่มีสิทธิ์** จัดการผู้ใช้, เพิ่ม/ลบอุปกรณ์, ดูรายงาน, หรือตั้งค่าระบบ

## 2. การเข้าสู่ระบบ

### ขั้นตอน:
1. เปิดเบราว์เซอร์ (Chrome / Edge แนะนำ) ไปที่ URL ของระบบ
2. คลิกปุ่ม **"เข้าสู่ระบบด้วย Google"**
3. เลือกบัญชี Google ที่มีสิทธิ์ Staff
4. ระบบจะนำไปยัง **หน้าหลัก** โดยอัตโนมัติ
5. คลิกเมนู **"Staff Panel"** ที่ Navigation หรือไปที่ `/staff`

> ⚠️ หากไม่เห็นเมนู Staff Panel ให้ติดต่อ Admin เพื่อตรวจสอบบทบาท

---

## 3. หน้า Dashboard ของ Staff

หน้าแรกของ Staff Panel แสดง **ภาพรวมงานที่รอดำเนินการ**:

### ข้อมูลที่แสดง:
- **คำขอยืมที่รออนุมัติ** — จำนวนคำขอใหม่ที่ยังไม่ดำเนินการ
- **อุปกรณ์ที่กำลังถูกยืม** — รายการที่ยืมอยู่ในปัจจุบัน
- **อุปกรณ์ค้างคืน** — รายการที่เลยกำหนดคืนแล้ว
- **การจองวันนี้** — การจองที่มีกำหนดใช้งานวันนี้

---

## 4. การจัดการคำขอยืม

**เส้นทาง:** Staff Panel → คำขอยืม (Loans)

### 4.1 การดูคำขอที่รออนุมัติ

1. ไปที่ **Staff Panel → คำขอยืม**
2. ระบบแสดงรายการคำขอแยกตามสถานะ:
   - **รอการอนุมัติ** — ต้องดำเนินการ
   - **อนุมัติแล้ว** — รออุปกรณ์ออก
   - **กำลังยืม** — ยืมอยู่ในปัจจุบัน
   - **คืนแล้ว** — ปิดคำขอแล้ว

### 4.2 การอนุมัติคำขอยืม

1. คลิกคำขอที่มีสถานะ **"รอการอนุมัติ"**
2. ตรวจสอบรายละเอียดคำขอ:
   - ชื่อผู้ขอยืม และภาควิชา
   - อุปกรณ์ที่ขอยืม (ชื่อ / Serial Number)
   - วันที่ยืม และ วันที่ / เวลาคืน
   - วัตถุประสงค์การยืม
3. ตรวจสอบว่าอุปกรณ์มีสถานะ **"ว่าง"**
4. คลิก **"อนุมัติ"** → ยืนยัน
5. ระบบแจ้งเตือนผู้ยืมอัตโนมัติ และเปลี่ยนสถานะอุปกรณ์เป็น **"ถูกยืม"**

### 4.3 การปฏิเสธคำขอยืม

1. คลิกคำขอที่ต้องการปฏิเสธ
2. คลิก **"ปฏิเสธ"**
3. พิมพ์ **เหตุผลการปฏิเสธ** (ระบบบังคับกรอก)
4. คลิก **"ยืนยัน"**
5. ระบบแจ้งเตือนผู้ยืมพร้อมเหตุผล

**ตัวอย่างเหตุผลที่ใช้บ่อย:**
- อุปกรณ์อยู่ระหว่างซ่อมบำรุง
- ช่วงเวลาทับซ้อนกับการจองอื่น
- ผู้ยืมมีอุปกรณ์ค้างคืนอยู่

## 5. การรับคืนอุปกรณ์

**เส้นทาง:** Staff Panel → รับคืนอุปกรณ์ (Returns)

นี่คือขั้นตอนสำคัญ — เมื่อผู้ยืมนำอุปกรณ์มาคืน Staff ต้องบันทึกในระบบทุกครั้ง

### ขั้นตอนรับคืนอุปกรณ์:

1. ไปที่ **Staff Panel → รับคืนอุปกรณ์**
2. ค้นหาคำขอยืมที่ต้องการรับคืน:
   - ค้นหาด้วยชื่อผู้ยืม หรือ
   - ค้นหาด้วยชื่อ / Serial Number ของอุปกรณ์
3. คลิกที่รายการที่ต้องการ
4. ตรวจสอบสภาพอุปกรณ์ก่อนรับคืน:
   - สภาพภายนอก (รอยขีดข่วน / ความเสียหาย)
   - อุปกรณ์เสริม (สายชาร์จ, กระเป๋า ฯลฯ)
5. คลิก **"บันทึกการคืน"**
6. หากมีความเสียหาย: ระบุใน **"หมายเหตุ"** ก่อนบันทึก
7. ยืนยัน → ระบบเปลี่ยนสถานะอุปกรณ์เป็น **"ว่าง"** อัตโนมัติ

> ✅ หลังบันทึกสำเร็จ: ระบบส่งการแจ้งเตือนให้ Admin และ Discord โดยอัตโนมัติ

## 6. การจัดการอุปกรณ์ค้างคืน

**เส้นทาง:** Staff Panel → ค้างคืน (Overdue)

### การดูรายการค้างคืน:

1. ไปที่ **Staff Panel → ค้างคืน**
2. ระบบแสดงรายการอุปกรณ์ที่เลยกำหนดคืน พร้อมข้อมูล:
   - ชื่อผู้ยืม / เบอร์ติดต่อ
   - อุปกรณ์ที่ค้างคืน
   - วันที่ / เวลาที่ควรคืน
   - จำนวนวัน/ชั่วโมงที่เกินกำหนด

### สิ่งที่ควรทำ:
1. ตรวจสอบรายการค้างคืนทุกวัน
2. ติดต่อผู้ยืมโดยตรง (โทร / LINE / อีเมล)
3. บันทึกหมายเหตุใน **"หมายเหตุการติดต่อ"**
4. หากผู้ยืมนำมาคืน → ดำเนินการที่ **"รับคืนอุปกรณ์"**

---

## 7. การจัดการการจอง

**เส้นทาง:** Staff Panel → การจอง (Reservations)

### ขั้นตอน:
1. ไปที่ **Staff Panel → การจอง**
2. ดูรายการการจองทั้งหมด แยกตามสถานะ:
   - **รอการอนุมัติ**
   - **อนุมัติแล้ว**
   - **ยกเลิก**
3. **อนุมัติการจอง**: คลิกรายการ → คลิก **"อนุมัติ"** → ยืนยัน
4. **ปฏิเสธการจอง**: คลิกรายการ → คลิก **"ปฏิเสธ"** → ระบุเหตุผล

---

## 8. การยืมอุปกรณ์สำหรับ Staff เอง

Staff สามารถยืมอุปกรณ์ผ่านหน้าหลักได้เช่นเดียวกับผู้ใช้ทั่วไป โดย **ได้รับการอนุมัติอัตโนมัติ** ไม่ต้องรอ Admin:

### ขั้นตอน:
1. ไปที่หน้า **"อุปกรณ์"**
2. ค้นหาอุปกรณ์ที่ต้องการยืม
3. คลิก **"ขอยืม"** หรือ **"จอง"**
4. กรอกข้อมูล: วันที่ยืม, วันที่/เวลาคืน, วัตถุประสงค์
5. ส่งคำขอ → **ระบบอนุมัติอัตโนมัติทันที**
6. ไปรับอุปกรณ์ตามที่ตกลง

## 9. การแก้ไขปัญหาเบื้องต้น

### ปัญหา: ไม่สามารถบันทึกการคืนได้
**วิธีแก้ไข:**
1. ตรวจสอบว่าเข้าสู่ระบบด้วยบัญชีที่มีสิทธิ์ Staff
2. รีเฟรชหน้าเว็บ (F5)
3. ลองค้นหาคำขอยืมใหม่อีกครั้ง
4. หากยังไม่ได้ → ติดต่อ Admin

### ปัญหา: คำขอยืมไม่ปรากฏในรายการ
**วิธีแก้ไข:**
1. ตรวจสอบว่าไม่ได้กรองสถานะไว้
2. รีเฟรชหน้าเว็บ
3. ตรวจสอบว่าคำขอส่งมาถูกต้องจากฝั่งผู้ใช้

### ปัญหา: ระบบแสดงข้อผิดพลาดขณะอนุมัติ
**วิธีแก้ไข:**
1. ตรวจสอบสถานะอุปกรณ์ว่ายังคง "ว่าง" อยู่
2. รีเฟรชหน้าเว็บแล้วลองใหม่
3. แจ้ง Admin พร้อมแจ้งข้อความ Error ที่พบ
